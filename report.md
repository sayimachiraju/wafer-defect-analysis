# Semiconductor Wafer Defect Analysis
### OVITO + SQLite Pipeline | Si Wafer Study

--- 
## Overview 
Computational study of lattice defects in silicon wafers using OVITO's python API and a SQLlite logging pipeline. Two defect regimes are investigated: 
1. Point defects (vacancies and interstitials) in monocrystalline Si as a function of implantation dose
2. Grain boundary disorder in bicrystalline Si as a function of misorientation angle 

--- 

## Methods
**Structure generation** - Silicon diamond cubic supercells (10x10x10 unit cells, 8000 atoms, a = 5.431 A) were build using ASE. Point defects were introduced by randomly removing (vacancies) and displacing a subset off-lattice (interstitials) at four vacancy fractions:
1%, 2%, 5%, 10%. Bicrystal structures were generated by splitting the supercell into two grains rotated ±θ/2 about the Z axis at misorientation angles of 15°, 30°, and 45°.

**OVITO analysis**
- Point defects: Wigner-Seitz defect analysis against a perfect Si reference structure
- Grain boundaries: IdentifyDiamond modifier with adaptive cutoff 

**Database** - All runs logged to SQLite ('defect_db.sqlite') with two tables:
'point_defect_runs' and 'grain_boundary_runs'. Defect metrics updated post-analysis.

---
## Results 
### Point defects
Defect density scales linearly with implantation done across all vacancy fractions. At 10% vacancy fraction, defect density reaches -3.3x10¹⁵ cm⁻², well above typical fab process thresholds (~10¹⁰–10¹² cm⁻²). Anneal temperature hand negligible effect in this study as annealing was not physically simulated - a noted limitation. 


| Vacancy fraction | Dose (cm⁻²) | Defect density (cm⁻²) |
|-----------------|-------------|----------------------|
| 1% | 2.71×10¹⁴ | ~3.4×10¹⁴ |
| 2% | 5.42×10¹⁴ | ~6.6×10¹⁴ |
| 5% | 1.36×10¹⁵ | ~1.7×10¹⁵ |
| 10% | 2.71×10¹⁵ | ~3.3×10¹⁵ |

### Grain boundaries
Disordered fraction increases monotonically with misorientation angle, consistent
with Read-Shockley theory for high-angle grain boundaries.

| Misorientation angle | Disordered fraction |
|---------------------|-------------------|
| 15° | 9.1% |
| 30° | 24.5% |
| 45° | 37.0% |

## Limitations & Future Work
- Annealing effects were not physically simulated; MD relaxation via LAMMPS would improve realism
- Only symmetric tilt boundaries were studied; asymmetric and twist boundaries are future targets
- Extending to GaAs or SiC wafers would require different interatomic potentials
- CALPHAD validation of defect formation energies is a natural extension
